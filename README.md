# analysing-twitter-data
Studying networks and NLP with twitter data


---
To activate this environment, use

    $ conda activate twitter-data

To deactivate an active environment, use

    $ conda deactivate

Install python package

    $ conda install -n twitter-data pip

Save a yml file with the necessary information to create a new env

    $ conda env export >> environment.yml

Create a environment using yml file

    $ conda env create -f environment.yml
